Mold

A fresh take on managing common development tasks like running tests, checking code quality, or building documentation.

Get the ready-to-use shell script here:

curl -sSfL xtfc.org/mold.sh | sh

mold.sh will download the latest Mold binary behind-the-scenes and proxy all arguments through to it. It's intended to be tracked by version control, so it's suitable for use in environments that don't already have Moldinstalled, like CI or your coworker's computer.
